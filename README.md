# ThreatLens

AI-powered threat detection and incident triage platform. Detects attacks (SQL injection,
XSS, brute force, traffic anomalies) against a web application in real time and uses the
Claude API to turn raw security alerts into actionable, OWASP-aligned incident reports.

> **Status:** under construction — foundation phase. Architecture, setup instructions, and a
> full walkthrough will land as the project progresses.

ThreatLens is a defensive security tool: it does not generate, launch, or assist real
attacks. It is intended for systems you own or are explicitly authorized to monitor.
